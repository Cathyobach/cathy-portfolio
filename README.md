# Cathy Portfolio Website - Export Package

This folder contains everything you need to deploy your portfolio website anywhere.

## What's Included

- `index.html` - Main HTML file
- `favicon.png` - Site icon
- `assets/` - All CSS, JavaScript, and images

## How to Deploy

### Option 1: Netlify (Recommended - Free)
1. Go to [netlify.com](https://netlify.com) and sign up
2. Drag and drop this entire folder onto the Netlify dashboard
3. Your site will be live in seconds with a free URL!

### Option 2: Vercel (Free)
1. Go to [vercel.com](https://vercel.com) and sign up
2. Create a new project and upload this folder
3. Deploy!

### Option 3: GitHub Pages (Free)
1. Create a new repository on GitHub
2. Upload all these files to the repository
3. Go to Settings > Pages > Select "main" branch
4. Your site will be live at `yourusername.github.io/repo-name`

### Option 4: Any Web Hosting
Upload all files to your web hosting via FTP or file manager. Make sure `index.html` is at the root level.

### Option 5: Local Preview
Open `index.html` directly in your browser to preview the site.

## Custom Domain
Most hosting platforms allow you to connect a custom domain (like cathyva.com). Check your hosting provider's documentation for instructions.

## Need Help?
The site is a static website - it works without any server-side setup. Just upload and go!
